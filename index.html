Guess tiles

Welcome to the "Guess the Tile" game! Your goal is simple: uncover two identical tiles in as few attempts as possible. Each pair has a unique number, but the tiles get shuffled with every new game start. Open the tiles, remember which numbers are behind which, and try to find all pairs as quickly as you can! Are you ready to test your memory?

Calendar+Weather

This is a simple web calendar that displays the weather for each day of the current month. The weather data is sourced from the OpenWeatherMap API. The calendar can fetch weather data for a city stored in the browser's local storage or for the user's current location (using geolocation). Additionally, there's a button to return to the main page of the site.

Calculator

An advanced calculator that combines standard arithmetic operations with functions like exponentiation, square root calculation, sine, and cosine. Easy to use and supports keyboard input.
